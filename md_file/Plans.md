# Plans.md

## 🗂 プロジェクト概要
**General Incorporated Association Authorized Cannabinoid (AC)**
日本のカンナビノイド市場における「正規通貫」「科学的根拠」「法令遵守」のスタンダードを確立するためのWebサイト構築。

精神作用成分の排除ではなく、「適正な輸入手続き」と「ISO17025準拠の検査」を経た安全な原料の普及を目的とする。

## 🚀 Execution Plan

- [x] プロジェクト初期セットアップ (`index.html` 作成)
- [x] **Criteria (認定基準) の修正**
    - ISO17025認証の対象を「検査機関/原料」に特定（事業者の認証ではない）。
- [x] **Zero Tolerance (排除方針) の修正**
    - 精神作用成分そのものの否定から、「不適正な通関」「危険ドラッグ」の排除へ文言変更。
- [ ] **Criteria構造の分離**
    - 「入会資格（事業者要件）」と「認定基準（製品要件）」を明確に分けて表示する。
- [ ] **Design Refinement**
    - モバイルレスポンシブ対応の強化
    - プレミアム感を高めるマイクロインタラクションの追加
- [ ] **SEO Optimization**
    - メタデータの最適化
    - OGP設定
- [ ] **Form Integration**
    - 入会申請フォームの実装（またはGoogle Formsへのリンク）

## 📜 Decision Log

### 2025-12-08: 基準と排除方針の明確化
*   **決定**: Criteria記述を「ISO17025認定機関による検査証明書を有する原料の使用」に変更。
*   **決定**: Zero Tolerance記述から「精神作用性成分」という単語を削除し、「適正な通関手続きを経ていない製品」に変更。
*   **決定**: 輸入通関基準を「許可証の保有」から「許可証に紐づく原料の使用」へ変更。
*   **決定**: 認定基準に「フルパネル検査（重金属、農薬、残留溶媒等）」のクリアを明記。
*   **理由**: 団体は精神作用の有無ではなく、法的な整合性とトレーサビリティ（原料と証明書の紐付け）を基準とするため。また、消費者の健康リスク（汚染物質）を排除するため。
